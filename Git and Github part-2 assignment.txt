1. run the command "git --version" in command prompt or git bash, for checking whether git is available on system or not.
2. git init
3. git config --global user.name = "UserName"
   git config --global user.email="user-email"
4. git add filename
5. git rm --cached filename
6. git commit -m "any suitable message"
7. git push origin main
8. clone means getting a copy of full project and pull means reflecting the recent changes into the cloned file
